AutoWeka Tutorial

发表于 2018-07-11

AutoWeka 使用教程

概述

在WEKA中有许多即选即用的机器学习算法，然而，这些算法及它们各自的参数都会显著地影响最后的结果，
各种不同算法与参数的组合数量也多到让人难以选择的地步。Auto-WEKA则是同时选择学习算法和它们参数，
让使用者不必一再尝试。Auto-WEKA借用贝氏最佳化（Bayesian optimization）的创新做法，
实现了完全自动化的方案。
我们希望Auto-WEKA可以帮助非专业使用者更有效率地找到适合他们数据的机器学习算法和其参数，
并且最终提升机器学习的效果。 —AutoWeka官网

AutoWeka前置环境

使用图形界面的前置环境

安装Java并且配置环境变量

  • 访问Java官网进行下载
  • 运行下载的安装程序，选择目录进行安装，假设安装到了C:\Program Files\Java\
  • 配置环境变量，设置JAVA_HOME=C:\Program Files\Java\jdk1.8.0_161
  • 在PATH中添加该环境变量%JAVA_HOME%\bin

安装Weka

  • 访问Weka官网进行下载，这里我们选择
    3.8稳定版。
  • 运行下载的安装程序，选择目录进行安装，这里我们安装到D:\Weka-3-8
  • 设置环境变量CLASSPATH，新增D:\Weka-3-8\weka.jar
  • 进入安装目录，双击Weka 3.8启动

安装AutoWeka

安装AutoWeka有两种方式，分别是通过Weka自带的包管理器在线安装（推荐）以及下载AutoWeka独立版进行安装。接下来我们将分别介绍。

在线安装
  • 启动Weka，点击Weka工具栏上Tools中的Package Manager
  • 选中AutoWeka进行安装，安装成功后重启Weka。
离线安装
  • 进入AutoWeka官网，下载最新版本2.6。
  • 进入Package Manager点击右上角的File/URL,选中下载的zip即可进行安装，安装完毕后重启weka。

使用AutoWeka

接下来，我们分别在GUI（图形界面）以及CLI（命令行）中以Weka软件下data目录中自带的鸢尾花Iris为例子，说明如何使用AutoWeka。

GUI方式

  • 打开weka,点击Explorer
  • 点击左上角的open file选择data下的iris.arff数据集。
  • 切换到Classify面板，点击左上角的Choose，在meta下选择AutoWeka分类器
  • 点击AutoWekaClassifier的参数，弹出参数设置窗口，其中主要属性为timeLimit，它代表AutoWeka为我们
    寻找最佳算法和参数的时间，如果短时间内得到的正确率较低，可以延长该参数。
  • 为了测试，我们将timeLimit改为1，即1分钟。左侧的Test options选择Use training set，点击Start。
  • 输出框中，我们可以看到很多内容，我们着重关注You can use the chosen classifier in your own code as follows:下的话

我们可以看到AutoWeka帮我们选择了weka.classifiers.functions.SMO，即SMO算法，参数为
-C 1.2047011958306952 -N 2 -M -K weka.classifiers.functions.supportVector.Puk -S 4.74388763709961 -O 0.21103679312955204
正确率98.6667%

  • 接下来我们使用SMO算法进行验证，右键点击Choose右侧算法名字的位置,选择Enter configuration
    输出算法的类路径和参数
    weka.classifiers.functions.SMO -C 1.2047011958306952 -N 2 -M -K "weka.classifiers.functions.supportVector.Puk -S 4.74388763709961 -O 0.21103679312955204"

CLI方式

  • 首先我们在命令行中进入到AutoWeka独立版（即上文的离线下载包）解压的目录，
  • 在命令行输入
    java -cp autoweka.jar weka.classifiers.meta.AutoWEKAClassifier -t data\iris.arff -timeLimit 15 -no-cv

-cp 参数后面是类路径，是指定给解释器到哪里找到你的.class文件，-t后面填写输入文件位置，默认目录为根目录，所以我们要将iris.arff拷贝到当前目录。
-no-cv防止WEKA将数据拆分为训练和测试集，这些行为Auto-WEKA在已经替它执行过了。

  • 在命令行执行中，常常会报警告[main] WARN Experiment - [WARN ] [PROCESS-ERR] 2018-07-11 17:00:35.728 [Thread-0] WARN ClassifierRunner - Training classifier (weka.classifiers.meta.RandomCommittee [-I, 5, -S, 1, -W, weka.classifiers.rules.OneR, --, -B, 3]) failed: Base learner must implement Randomizable!
    并不妨碍执行，可以无视。
  • 最终在命令行上输出结果。
